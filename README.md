# dotfiles

## Introduction

shihtiy's config files

## Structures

The dotfiles includes the dotfiles of the following tools:

- VIM
- Neovim
- ZSH
- tmux
- Git
- Mac
- BASH

## Getting Start

```bash
git clone https://github.com/shihtiy-tw/dotfiles.git

make install

make init
```
